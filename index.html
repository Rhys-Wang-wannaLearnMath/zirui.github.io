I led the project to implement a Redis-like caching system in Rust with a modular architecture. We refactored the storage module for better maintainability, built an asynchronous connection layer using Tokio, and designed a flexible logging system with configurable levels and colored output. Mini-Bench: Mobile LLM Performance Evaluation

I developed Mini-Bench, an evaluation framework for mobile LLMs targeting QA, comprehension, and sentiment analysis tasks. I curated datasets from sources like Natural Questions, TriviaQA, and SQuAD using Claude, implemented a weighted scoring system with Z-score normalization, and built an automated pipeline for various question formats. Ultimately, I evaluated eight edge LLMs, including the Qwen and Llama series, to gain insights for deployment optimization.
